# CS769-HW5

## Setup and Installation

1. Clone the repository:
```bash
git clone https://github.com/Zefan-Cai/CS769-HW5.git
cd CS769-HW5
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

### Running Evaluations
To run the evaluations:
